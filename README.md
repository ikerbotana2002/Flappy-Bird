# Flappy-Bird
# Proyecto de Animación en Unity

Este repositorio contiene el proyecto de Unity, junto con la memoria en PDF y un vídeo de demostración del funcionamiento del juego.


## Descripción

El proyecto consiste en una escena de animación creada con Unity, donde se muestran diferentes técnicas de animación aplicadas a personajes y/o elementos del entorno.  
Incluye tanto los recursos necesarios para abrir el proyecto en Unity como documentación escrita y un vídeo explicativo del funcionamiento general.

## Contenido del repositorio

- `Memoria-Animacion.pdf`: Documento con la memoria del proyecto, explicación teórica y técnica.
- `Assets/`, `ProjectSettings/`, etc.: Carpetas estándar del proyecto de Unity con todo el código y recursos.
- `Video Flappy Bird.mp4`: Vídeo que muestra el funcionamiento del proyecto y las animaciones implementadas.
- `README.md`: Este archivo de documentación.

## Tecnologías utilizadas

- **Motor**: Unity
- **Lenguaje**: C# para los scripts.
- **Herramientas adicionales**:  
  - Sistema de animación de Unity (Animator, Animation Clips, etc.).
